# MonkFighter2

Standalone World of Warcraft addon for Windwalker Monk.

## Features

- custom sound effects for supported abilities
- movable combo counter HUD
- separate Combo Strikes tracking list and sound mapping list
- talent/replacement spell support in the options UI
- optional combo persistence outside combat
- configurable out-of-combat HUD hide delay
- `COMBO BREAKER!` sound on repeated combo-tracked casts

## Install

Drop this `MonkFighter2/` folder into:

`World of Warcraft/_retail_/Interface/AddOns/`

## Commands

- `/mf2`
- `/mf2 unlock`
- `/mf2 lock`
- `/mf2 reset`

## Notes

- Options are stored in `MonkFighter2DB`.
- Sound mappings are configurable from the addon options panel.
